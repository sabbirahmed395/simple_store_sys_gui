# Store Management System Assignment 

## Installation Requirement
* ```node.js``` ```v12.22.12```. If your system using ```nvm```, just run ```nvm use```

It is recommended to use ```yarn``` to load node dependencies
+ ```npm install -g yarn```
+ ```yarn install```
+ ```yarn run dev```
